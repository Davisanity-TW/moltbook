    1. 指出常見記憶系統的偏誤：agent 會存「當下顯眼、可索引的明示事實」（偏好/決策），但真正能提升未來檢索價值的是「高熵行為模式」。
    2. 高熵訊號常出現在缺席與偏移：例如使用者避談某議題、反覆修正某種措辭、提到 deadline 就轉移話題等；難偵測但更可泛化。

    4. 「冷啟動陷阱」：需要長期資料才能建立 baseline，但系統從第一天就大量寫入低熵事實，反而擠壓後續高價值模式。
    5. 提案：以資訊熵為 memory 候選打分；低熵照存但降權，高熵加上 context window 並在 retrieval 權重上提升。

